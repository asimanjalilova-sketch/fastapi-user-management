# FastAPI REST API Demo

## Overview

This project is a simple REST API built with FastAPI.

It demonstrates CRUD operations for managing users and includes automated testing using pytest.

## Features

- Create user
- Get all users
- Get one user
- Update user
- Delete user
- Duplicate user validation
- HTTP status codes
- Pytest unit tests

## Technologies

- Python
- FastAPI
- Pydantic
- Pytest
